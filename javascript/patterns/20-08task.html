PART-01 Arithmetic Operators

  • 1. Declare two numbers a = 12, b = 5. Print sum, difference, product, quotient, and remainder.
  • 2. Use the exponentiation operator to calculate 5² and 3³.
  • 3. Create a variable num = 20. Apply increment (++) and decrement (--) operators step by step.


    PART-02: Assignment Operators

  • 4. Create a variable x = 50. Use +=, -=, *=, /=, %= operators on x. Print after each step.
  • 5. Start with score = 100. Add 10 using +=, subtract 20 using -=, then divide by 2 using /=. Print final score.


    PART-03: Comparison Operators

  • 6. Declare age1 = 25, age2 = "25". Compare using == and ===. Print results
  • 7. Write a program to check if marks = 35 is greater than or equal to 40. If true → print “Pass”, else → “Fail”.
  • 8. Take p = 15, q = 30. Print results of p > q, p < q, p>= q, p <= q, p !=q, p !==q.


    PART-04: Logical Operators

  • 9. Create isStudent = true, hasIDCard = false. Use &&, ||, and ! to check exam hall eligibility.
  • 10. Check if temperature = 32. If between 30 and 40 → print “It’s Hot”, else “Normal”.
  • 11. Create username = "admin", password = "1234". If both are correct → print “Login Successful”, else → print “Login Failed”.


    PART-05: Mixed-Type Operator Tasks

  • 12. Add 10 + "20". Print result and type.
  • 13. Subtract "50" (string) from 20. Print result.
  • 14. Multiply true with 5. Print result.
  • 15. Divide 0 / 10 and 10 / 0. Print both results.
