WEEK 01 TASKS

    Part A: Basics & Variables (1-10)

  1. Print "Hello JavaScript" in console.
  2. Declare a variable name and store your name, then print it.
  3. Swap two numbers using a temporary variable.
  4. Swap two numbers without using a temporary variable.
  5. Declare constants for PI and gravity, then display them.
  6. Show difference between let, const, and var.
  7. Store your age in a variable and check its type.
  8. Concatenate first name and last name into a full name.
  9. Declare variables of all basic data types (string, number, boolean, null, undefined) and display them.
  10. Reassign a let variable and explain why const cannot be reassigned.

    Part B: Data Types & Conversion (11-15)

  1. Convert a number to a string using String() and .toString().
  2. Convert a string "123" to a number using Number() and parseInt().
  3. Show the difference between == and ===.
  4. Check if "Hello" is of type string.
  5. Demonstrate typeof operator with all data types.

    Part C: String Methods (16-25)

  1. Find the length of "JavaScript".
  2. Convert "hello world" into uppercase.
  3. Convert "HELLO WORLD" into lowercase.
  4. Extract the first 4 characters from "Programming".
  5. Replace "Java" with "Type" in "JavaScript".
  6. Find the index of "Script" in "JavaScript".
  7. Check if "world" exists in "Hello world".
  8. Trim extra spaces from " JS Basics ".
  9. Split "Apple, Banana, Mango" into an array.
  10. Concatenate two strings using .concat().

    Part D: Number & Math Methods (26-35)

  1. Round 4.7 using Math.round().
  2. Use Math.floor(9.8) and Math.ceil(9.2).
  3. Generate a random number between 0 and 1.
  4. Generate a random number between 1 and 100.
  5. Find the maximum and minimum from (10, 5, 20, 3).
  6. Find square root of 64.
  7. Find absolute value of -25.
  8. Convert a float 45.6789 into 2 decimal places.
  9. Demonstrate isNaN() with different inputs.
  10. Use Math.pow(2, 5) to calculate 2⁵.

    Part E: Operators (30-45)

  1. Add two numbers using + operator.
  2. Use modulus % operator to find remainder of 25 ÷ 4.
  3. Demonstrate ++ pre-increment and post-increment.
  4. Use comparison operators to check if 10 > 5.
  5. Check if two strings "JS" and "JS" are equal.
  6. Use logical && to check if 5 > 2 and 10 > 5.
  7. Use logical || to check if either condition is true.
  8. Use logical ! to reverse a boolean value.
  9. Demonstrate += operator with a variable.
  10. Write a small expression combining arithmetic and logical operators.

    Part F: Conditionals (46-50)

  1. Write a program to check if a number is positive, negative, or zero.
  2. Check if a number is even or odd.
  3. Check if a person is eligible to vote (age ≥ 18).
  4. Write a program to find the largest of two numbers.
  5. Write a program to find the largest of three numbers.
