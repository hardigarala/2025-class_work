Array and Object

Array - 10 Practical Tasks

  1. Create an array of numbers and find the sum of all elements.
  2. Find the maximum and minimum numbers from an array.
  3. Reverse an array without using the built-in reverse() method.
  4. Remove duplicate elements from an array.
  5. Sort an array in ascending and descending order.
  6. Merge two arrays and remove duplicates from the merged array.
  7. Count how many times a specific element appears in an array.
  8. Find the second largest element in an array.
  9. Check if all elements in an array are even numbers.
  10. Rotate an array by n positions (e.g., shift elements left by 2).

Object - 10 Practical Tasks

  1. Create an object representing a student with name, age, and marks. Print all properties.
  2. Update the value of a property in an object.
  3. Delete a property from an object.
  4. Count the number of properties in an object.
  5. Write a function to check if a property exists in an object.
  6. Convert an object into an array of keys and an array of values.
  7. Merge two objects into a single object.
  8. Create an array of objects (students) and filter students with marks above 80.
  9. Sort an array of objects by a specific property (e.g., age).
  10. Find the object with the highest value for a given property (e.g., max salary from employee objects).
